# LeapYear Merge Bot

## Quickstart

1. `scripts/install-system-deps.sh`
1. `stack build`
1. `source .env`
1. `stack exec merge-bot`
1. Go to `smee.io` and start a new channel
1. `smee --url https://smee.io/<token>`
1. Run configuration steps pointed to `https://smee.io/<token>`
